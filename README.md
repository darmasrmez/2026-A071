# How to use it

1. Clone the repository
```bash
git clone https://github.com/darmasrmez/2026-A071.git
```
2. Create a virtual environment (we recommend using uv)
```bash
uv venv
```
3. Install the dependencies
```bash
uv sync
```
4. Edit the params.yml file with the desired model and size

5. Run the application
```bash
uv run main.py
```
